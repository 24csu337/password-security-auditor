# Password Security Auditor

## Overview

Password Security Auditor is a Python application that evaluates password strength.

The program checks:

- Password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

It then:

- Calculates a score
- Assigns a strength rating
- Provides suggestions for improvement

## Technologies Used

- Python

## Features

- Password strength analysis
- Security scoring system
- Password rating system
- Improvement recommendations
